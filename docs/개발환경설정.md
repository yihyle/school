# LearnHub 개발 환경 설정 가이드

## 사전 요구사항

| 항목 | 버전 | 설치 방법 |
|------|------|-----------|
| Java | 17 이상 | `brew install openjdk@17` |
| Node.js | 18 이상 | `brew install node` |
| MySQL | 8.0 이상 | `brew install mysql` |
| Git | - | `brew install git` |

---

## 1. 저장소 클론

```bash
git clone https://github.com/yihyle/school.git
cd school
```

---

## 2. MySQL 데이터베이스 설정

### MySQL 실행
```bash
brew services start mysql
```

### 데이터베이스 생성
```bash
mysql -uroot -p -e "CREATE DATABASE IF NOT EXISTS learnhub CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
```

### 접속 정보 (기본값)
| 항목 | 값 |
|------|-----|
| Host | localhost |
| Port | 3306 |
| Database | learnhub |
| Username | root |
| Password | 1234 |

> 비밀번호가 다를 경우 `backend/src/main/resources/application.yml`에서 `spring.datasource.password` 값을 수정하세요.

---

## 3. 백엔드 실행 (Spring Boot)

```bash
cd backend
./gradlew bootRun
```

- 서버 주소: http://localhost:8080
- Swagger UI: http://localhost:8080/swagger-ui.html
- 최초 실행 시 Flyway가 자동으로 테이블 생성 및 시드 데이터를 삽입합니다.

---

## 4. 프론트엔드 실행 (Next.js)

### 환경변수 파일 생성

`frontend/.env.local` 파일을 생성하세요 (Git에 포함되지 않음):

```
NEXT_PUBLIC_API_URL=http://localhost:8080
```

### 의존성 설치 및 실행

```bash
cd frontend
npm install
npm run dev
```

- 서버 주소: http://localhost:3000

---

## 5. Git에 포함되지 않는 파일 목록

| 파일/폴더 | 위치 | 설명 | 생성 방법 |
|-----------|------|------|-----------|
| `.env.local` | `frontend/` | 프론트엔드 환경변수 | 위 4번 참고하여 수동 생성 |
| `node_modules/` | `frontend/` | npm 패키지 | `npm install` 실행 시 자동 생성 |
| `.next/` | `frontend/` | Next.js 빌드 캐시 | `npm run dev` 또는 `npm run build` 시 자동 생성 |
| `.gradle/` | `backend/` | Gradle 캐시 | `./gradlew` 실행 시 자동 생성 |
| `build/` | `backend/` | 빌드 산출물 | `./gradlew build` 시 자동 생성 |

---

## 6. 빌드 확인

### 백엔드 컴파일 확인
```bash
cd backend
./gradlew compileJava
```

### 프론트엔드 빌드 확인
```bash
cd frontend
npm run build
```

---

## 7. Docker를 이용한 MySQL 실행 (선택)

로컬에 MySQL이 없는 경우 프로젝트 루트의 `docker-compose.yml`을 사용할 수 있습니다.

```bash
docker compose up -d
```

> Docker 사용 시 `application.yml`의 password를 `root1234`로 변경하세요.  
> `docker-compose.yml`에 설정된 `MYSQL_ROOT_PASSWORD`가 `root1234`입니다.

---

## 8. 전체 실행 순서 요약

```bash
# 1. MySQL 실행 확인
mysql -uroot -p1234 -e "CREATE DATABASE IF NOT EXISTS learnhub;"

# 2. 터미널 1: 백엔드
cd backend && ./gradlew bootRun

# 3. 터미널 2: 프론트엔드
cd frontend
echo "NEXT_PUBLIC_API_URL=http://localhost:8080" > .env.local
npm install
npm run dev

# 4. 브라우저에서 http://localhost:3000 접속
```
