## 이슈/해결
- **Spring Initializr 버전 오류**: `bootVersion=3.4.5` 요청 시 400 Bad Request 발생
  - 원인: Spring Initializr 호환 범위가 `>=3.5.0`으로 변경됨
  - 해결: `bootVersion=3.5.0`으로 변경하여 재요청

## 학습 포인트
- Spring Initializr는 지원 버전 범위가 주기적으로 변경되므로 버전 명시 시 공식 사이트에서 최신 지원 범위 확인 필요
- Next.js App Router 구조에서 `src/` 디렉토리 없이 `app/` 을 루트로 사용하는 방식 적용
- Colima 기반 Docker 환경에서 아키텍처 불일치 문제가 발생할 수 있음 (Day 2에서 해결)
