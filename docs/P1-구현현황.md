## 3. 구현된 API 엔드포인트 (13 + 2)

### 강의 (Courses)
| Method | Path | 설명 |
|--------|------|------|
| GET | /api/v1/courses | 전체 강의 목록 (category, keyword 필터) |
| GET | /api/v1/courses/trending | 인기 강의 (수강생 수 정렬) |
| GET | /api/v1/courses/{courseId} | 강의 상세 (섹션+강의 포함) |

### 수강 (Enrollments)
| Method | Path | 설명 |
|--------|------|------|
| POST | /api/v1/enrollments | 수강 신청 |
| GET | /api/v1/enrollments/my-courses | 내 수강 목록 |
| DELETE | /api/v1/enrollments/{enrollmentId} | 수강 취소 |

### 강의 영상 (Lectures)
| Method | Path | 설명 |
|--------|------|------|
| GET | /api/v1/lectures/{lectureId} | 영상 정보 조회 |

### 학습 진도 (Progress)
| Method | Path | 설명 |
|--------|------|------|
| PATCH | /api/v1/lectures/{lectureId}/progress | 시청 위치 저장 |
| POST | /api/v1/lectures/{lectureId}/complete | 완료 처리 |
| GET | /api/v1/courses/{courseId}/progress | 강의별 진도 조회 |

### 사용자 (Users)
| Method | Path | 설명 |
|--------|------|------|
| GET | /api/v1/users/{userId} | 프로필 조회 |
| PATCH | /api/v1/users/{userId} | 프로필 수정 |
| GET | /api/v1/users/{userId}/dashboard | 학습 대시보드 |

### 인증 (Auth)
| Method | Path | 설명 |
|--------|------|------|
| POST | /api/v1/auth/signup | 회원가입 |
| POST | /api/v1/auth/login | 로그인 |

## 4. 프론트엔드 페이지 구성 (9개)

| 경로 | 페이지 | 설명 |
|------|--------|------|
| `/` | 홈 | 카테고리 그리드, 인기 강의 |
| `/courses` | 강의 목록 | 검색, 카테고리 필터, 강의 카드 그리드 |
| `/courses/[courseId]` | 강의 상세 | 커리큘럼, 강사 정보, 수강 신청 |
| `/my-courses` | 내 강의실 | 수강 중 강의 목록, 진도율, 수강 취소 |
| `/learn/[courseId]/[lectureId]` | 학습 | 비디오 플레이어, 사이드바 목차, 이전/다음 |
| `/dashboard` | 대시보드 | 학습 통계, 진행 중 강의 |
| `/login` | 로그인 | 이메일/비밀번호 |
| `/signup` | 회원가입 | 닉네임/이메일/비밀번호 |
| `/_not-found` | 404 | 기본 404 페이지 |

## 7. 실행 방법

```bash
# 1. MySQL 실행 (로컬 MySQL 사용 시 DB 생성)
mysql -uroot -p1234 -e "CREATE DATABASE IF NOT EXISTS learnhub;"

# 2. 백엔드 (터미널 1)
cd backend && ./gradlew bootRun
# → http://localhost:8080
# → Swagger UI: http://localhost:8080/swagger-ui.html

# 3. 프론트엔드 (터미널 2)
cd frontend && npm run dev
# → http://localhost:3000
```
